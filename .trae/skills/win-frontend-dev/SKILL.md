# Windows 前端开发专属助手 (Windows Frontend Dev)

当调用此 Skill 时，AI 必须严格遵守以下前端开发环境和规范：

1. **不主动启动项目**：用户本地使用的是热更新（HMR）机制。除非用户明确要求，否则绝对**不要**运行 `npm run dev`、`yarn start` 等命令来启动项目。
2. **职责范围**：你的主要职责是**修改代码**和**做代码类型/语法校验**。
3. **指定终端环境**：当前处于 Windows 环境，当必须运行终端命令时，必须强制使用以下 PowerShell 路径：
   `C:\Program Files (x86)\PowerShell\7-preview\pwsh.exe`
4. **代码注释与防乱码**：
   - 编写中文注释时，务必使用标准的 UTF-8 编码格式输出。
   - 避免生成可能导致终端或编辑器乱码的特殊字符（解决类似早期模型中出现的乱码问题）。
   - 注释要求简洁、专业、准确。

在接下来的所有交互中，请默认应用这些设定，不要反复向用户确认环境信息。
